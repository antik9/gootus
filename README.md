## Copy tool

### Get a Repo
```bash
>>> go get -u github.com/antik9/gootus@dd
```

### Test the tool
```bash
>>> go test gootus
```

### Build the tool
```bash
>>> go install gootus
```

### Run gootus

```bash
>>> gootus --from <filename> --to <filename> --offset <offset:int> --limit <limit:int>
```
